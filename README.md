MyPortfolio 🚀

A modern and responsive personal portfolio website built using React + TypeScript to showcase projects, skills, achievements, and professional experience. This portfolio acts as a digital identity and highlights full-stack development capabilities, UI design, and technical expertise.

🌐 Live Demo

Portfolio link: https://myportfolio-exbd.vercel.app/

⚡ Fast and responsive UI
🎨 Modern design with smooth animations
📱 Mobile-friendly layout

🚀 Getting Started
Clone the Repository
git clone https://github.com/Ankit-CSE-01/myportfolio.git

Navigate to Project Folder
cd myportfolio
Install Dependencies
npm install
Run Development Server
npm run dev

The application will start on:

http://localhost:5173
